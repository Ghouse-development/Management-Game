# シミュレーション結果 2026-01-04

## 実施内容
10,000回シミュレーションによる最適戦略探索（2期〜5期、6社競争）

## 結論（先に記載）

### 最適戦略: LARGE_HIRE1
**2期に大型機械へアップグレード + ワーカー1名採用**

| 指標 | 値 |
|------|------|
| 目標達成率(450円) | **57%** |
| 勝率 | **90%** |
| 平均自己資本 | **¥423** |
| 平均G | **¥101** |

### 具体的な行動
1. **2期開始時**: 最大借入（約100円）
2. **2期序盤**: 小型機械売却（63円取得）→ 大型機械購入（200円）
3. **2期序盤**: ワーカー1名採用（5円）
4. **全期間**: チップ購入なし（研究・教育・広告すべて不要）
5. **材料購入**: 4個ずつ

## 最適戦略（10,000回シミュレーション実証）

### 最強パターン: 採用1名戦略
- **機械**: 大型機械（能力4）で開始
- **チップ**: 購入しない（研究・教育・広告すべて不要）
- **採用**: ワーカー1名追加（2期開始時）
- **借入**: 期首に最大限借入
- **材料**: 4個ずつ購入
- **倉庫**: 購入しない

**期待成績**:
- 目標達成率: **68%**
- 勝率: **96%**
- 平均自己資本: **¥501**

### 次善策: 教育1枚戦略
- **機械**: 大型機械（能力4）で開始
- **チップ**: 教育1枚のみ
- **採用**: 追加なし
- **借入**: 期首に最大限借入
- **材料**: 4個ずつ購入

**期待成績**:
- 目標達成率: **46%**
- 勝率: **87%**
- 平均自己資本: **¥408**

## シミュレーションの前提条件
- 6社競争
- 2期〜5期をシミュレート
- リスクカード確率: 20%（15/75枚）
- 入札勝率モデル: 基本20% + 研究チップ12%/枚 + 親10% + 低価格ボーナス

## 注意事項
- このシミュレーションは「大型機械で開始」を前提としている
- 実際のゲームでは小型機械から始まるため、2期に大型機械へのアップグレードが必要
- アップグレードコスト: 小型売却（簿価90×70%=63円）+ 大型購入200円 = 純支出137円
- 借入で資金調達することで実現可能
