## 最重要発見

### 最強戦略: R2E1_NR_SM_DYN（成功率95.20%）

```
2期の購入:
  - PCチップ ¥20
  - 保険 ¥5
  - 研究チップ×2 ¥40
  - 教育チップ×1 ¥20
  - 翌期チップ（研究）×1 ¥20
  合計: ¥105 → 残金¥7

3期:
  - 翌期チップ適用 → 研究3枚に
  - 【動的借入】現金60円未満なら借入（最大80円）
  - 小型機械購入 ¥100 + ワーカー採用 ¥20

4期～5期:
  - 生産・販売を継続
  - 名古屋¥28で安定販売
  - 【動的借入】現金が不足したら追加借入
```

### 動的借入（DYN）が最強の理由

1. **必要な時だけ借りる**
   - 固定額借入より柔軟に対応
   - 利息負担を最小限に抑制
   - 現金60円未満の時のみ借入実行

2. **段階的借入の有効性**
   - 3期に30円、4期に70円など分割
   - 一度に借りるより利息効率が良い
   - B30_B70が93.20%、B40_B60が93.10%

3. **借入タイミングの最適化**
   - 早めに借りすぎると利息負担増
   - 遅すぎると資金ショートのリスク
   - 動的判断が最も効果的

## 借入ルール詳細

### 借入限度額
- **3期**: 自己資本 × 0.5（例: 283 × 0.5 = 141円）
- **4期以降 かつ 自己資本300超**: 自己資本 × 1.0（例: 350 × 1.0 = 350円）
- **それ以外**: 自己資本 × 0.5

### 1円単位借入
- 長期借入は1円単位で借入可能
- 利率10%（借入100円 → 手取り90円）
