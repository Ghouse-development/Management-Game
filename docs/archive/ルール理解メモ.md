## ルール体系（全体構造）

MGには以下のルール分類がある：

### Aルール（意思決定時のアクション）
- **タイミング**: 意思決定カードを引いたときにできる
- **項目数**: 9項目

#### A-1. 材料購入
| 項目 | 内容 |
|------|------|
| 単価 | 市場価格（海外は¥16固定） |
| 能力・効果 | 2期の1周目のみ3個まで |
| 注意 | マーケットをまたがって購入可能、別行に記入 |
| シニアルール | 製造能力の範囲内（教育チップ含む） |

#### A-2. 完成投入
| 項目 | 内容 |
|------|------|
| 単価 | 完成¥1、投入¥1 |
| 能力・効果 | 製造能力の範囲内（教育チップ含む） |
| 注意 | 完成から先に処理、仕掛品は10個まで |

**製造フロー**: 材料 → 投入(¥1) → 仕掛品 → 完成(¥1) → 商品

#### A-3. 商品販売
| 項目 | 内容 |
|------|------|
| 細目 | 入札、換金市場、海外投売り(¥16) |
| 能力・効果 | 販売ルールの通り、複数マーケットOK |

#### A-4. 商品仕入
| 項目 | 内容 |
|------|------|
| 概要 | 他社からの仕入れ（自社製造せず購入） |
| シニアルール | 販売能力の範囲内 |

### Cルール（運用のルール）

#### C-1. 時間
- 基本時間: 44分（5人）、前半22分+休憩5分+後半22分
- 6人: +3分、4人: -3分
- 実力に応じて延長・短縮あり

#### C-2. 第2期
- 時間: 33分（前半11分+休憩5分+後半22分）
- 終了: 行が一杯→その組終了、残りは後3分
- **2期のみ除外**: 得意先倒産(¥30)、返品(-1個-¥20)

#### C-3. 行数
| 期 | 行数 |
|-----|------|
| 2期 | 20行 |
| 3期 | 30行 |
| 4期 | 34行 |
| 5期 | 35行 |

#### C-4. 反時計
- 進行方向: 反時計まわり（左まわり）
- 1周目: 3個まで（ジュニア=2期）
- 2周目〜: 自由（資金の範囲内）

#### C-5. ゆっくり
- 第2期: 前の人が「残高」を書いてから、次の人がカードを引く

#### C-6. 戦略チップ
- 丸いチップ（赤・青・黄・緑）は売れない
- 一度購入したら現金化不可

#### C-7. シニアの目途
- 第2期: PQ 300以上でシニア
- 第3期: PQ 700以上でシニア
- ベテランは別途決定

#### C-8. 移動時の注意
- 物理ゲーム用ルール（デジタル版では不要）

#### C-9. 埋めない
- 市場の個数を満杯にしない（売れなくなる）
- 物理ゲーム用ルール

#### C-10. 本籍
- 自己資本グラフ・STRAC表は元の位置で続ける
- 物理ゲーム用ルール

#### C-11. グラフ
- グラフは太マジックできれいに書く
- 必ず数値と名前を書く
- 物理ゲーム用ルール

#### C-12. 現金残
- ゲーム中、現金残高がマイナスになるとき → インストラクターを呼ぶ
- 決算時、個数が合わないとき → インストラクターを呼ぶ
- **デジタル版**: 現金マイナス防止チェック、決算時の個数整合性チェックが必要

#### C-13. 決算時（資金不足対策）
- 短借りは100円単位
- 短借り100円 → 支払利息20円 + 返済20円 = **実質60円**手に入る
- 少額なら50円単位でも可
- ※借入時手取り80円だが、同期末に返済20円必要なので実質60円

#### C-14. 表彰者の条件
1. 4期5期と連続してシニアを2回やっている
2. 終了時、A++、S++である
3. 4期600超分社は最優先

#### C-15. 5行コール
- あと5行になったら「あと5行」とコールする
- 突然「ハイ終わり！」と言わない
- **デジタル版**: 各社個別に残り5行到達時に警告表示

#### C-16. 三つのダメ
1. 追い越しはダメ（ターン順を守る）
2. 残高未記入はダメ（残高を書いてから次へ）
3. 入札締切りはダメ（入札を急かさない）

#### C-17. 20/30/60の法則
- **第2期のみ免除**:
  - 返品 -1個 -20円
  - 得意先倒産 30円
- **全期通して**: 得意先倒産2回 60円 は存在しない

#### C-18. 期首金融
- 期首の金利を払い、コンピュータと倉庫を買ってマイナスになった人
- 長借100円（金利10円、手取り90円）が可能

#### C-19. 債務超過
- 自己資本マイナスの人は:
  - 期首金利免除
  - 2行目で長借100円（金利10円、手取り90円）が可能

#### C-20. 5個まで（ラッキーカード独占販売）
- すべてのラッキーカード（赤・青・黄）による独占販売は**最高5個まで**
- **教育成功（黄）**: 教育チップ所持で1個32円×5個まで（販売能力内）
- **研究開発成功（青）**: 研究開発チップ1枚につき2個、1個32円（仕入不可）
- **商品の独占販売（赤）**:
  - セールスマン1人につき2個まで、1個32円
  - 広告チップ1枚につき2個まで空いた市場へ（仕入可、割増なし）

#### C-21. 公正取引
- グループ全員の平均単価が「30円」台のとき
- → 公正取引委員会立入りとして**特損50円**計上

#### C-22. MGの哲学
- 楽しくなければ、MGでない
- 美しくなければ、MGでない
- 効果がなければ、MGでない

### 販売ルール

#### 販売ルール1（基本）
- 販売は市場で自由競争（入札制）
- 意思決定カードで販売宣言した人が「親」
- 親は空いている市場に、販売能力の範囲内で何個でも宣言可能
- 最高プライス = 親が宣言した市場のプライス
- 最も安い価格をつけた企業から順番に落札
- **価格競争力**: 親は常に+¥2、研究開発チップ1枚につき+¥2

**同価格時の優先順位**:
1. 研究開発チップの多い企業
2. 研究開発チップが同数なら親
3. 該当なしなら再入札（価格ダウン必須）
4. 最後はジャンケン

#### 販売ルール2（複数市場）
1. 親が複数市場に出す
2. 子が応札
3. 親が「同時入札」か「別々入札」を宣言
4. 子は降りてOK、別市場への変更は不可
5. 同時入札の場合、最低市場のPが上限

#### 販売ルール3（価格の種類）
- ①見せる（表示価格）: プライスカードで提示
- ②言う（口頭価格）: 競争で使う価格（親+¥2、研究開発1枚+¥2）
- ③書く（記帳価格）: 帳簿記録＝常に市場価格
- **重要**: 競争価格と記帳価格は異なる

### シニアルール（3期以降適用）

#### シニアルール1. 個数制限
**チップ関連**:
- チップは全て**40円**（特急料金）
- 使ったものは皆パー（返却）
- 次繰りは20円

**仕入制限**:
- 仕入は製造能力の範囲内
- 教育チップも効く

**在庫制限**:
- 材料・仕掛品・製品、各**10個まで**
- 11個以上は無災害倉庫（各20円）必要

**減価償却**:
- **2倍**（通常1割→2割、例: 26.40）
- 機械売却は期首簿価の**7割**

#### シニアルール2. サイコロ
3期・4期・5期の期首に、前期PQトップがサイコロを振る。

| サイコロ | 市場閉鎖 | 人件費倍率 | 大阪上限価格 |
|---------|---------|-----------|-------------|
| 1 | 仙台 | 1.1倍 | ¥21 |
| 2 | 仙台 | 1.1倍 | ¥22 |
| 3 | 仙台 | 1.1倍 | ¥23 |
| 4 | 仙台・札幌 | 1.2倍 | ¥24 |
| 5 | 仙台・札幌 | 1.2倍 | ¥25 |
| 6 | 仙台・札幌 | 1.2倍 | ¥26 |

※人件費は四捨五入

**人件費の計算（期末支払い）：**

| 期 | 基本単価 | サイコロ1〜3 (×1.1) | サイコロ4〜6 (×1.2) |
|----|---------|---------------------|---------------------|
| 2期 | ¥22 | - | - |
| 3期 | ¥24 | ¥26 / ¥13 | ¥29 / ¥14 |
| 4期 | ¥26 | ¥29 / ¥14 | ¥31 / ¥16 |
| 5期 | ¥28 | ¥31 / ¥15 | ¥34 / ¥17 |

※左が通常単価、右が期中最大人員単価（半額）

**人件費の内訳：**
- ワーカー × 単価
- 機械 × 単価
- セールスマン × 単価
- 期中最大人員 × 半額単価

**計算例（3期、サイコロ6、ワーカー1名・機械1台・セールスマン3名）：**
- ワーカー: 1 × ¥29 = ¥29
- 機械: 1 × ¥29 = ¥29
- セールスマン: 3 × ¥29 = ¥87
- 期中最大人員: 4 × ¥14 = ¥56
- **合計: ¥201**

#### シニアルール3. 研発6枚
- グループ内に1人でも**研究開発次繰り6枚以上**の人がいる組では、サイコロで研究開発の成功失敗を決める
- **奇数** = 成功
- **偶数** = 凍結（次繰り）
- ジュニアからシニアに上がる人はサイコロ不要

#### シニアルール4. 研究開発（降格時）
- シニアからジュニアに下がる人は、研究開発**3枚まで**
- 超過分は次繰り（凍結）

#### シニアルール5. 分社
- 自己資本**600**に達したら、分社
- その到達自己資本を新資本金として再スタート
- **無借金**でリセット

#### シニアルール6. 倉庫
- 倉庫は原則として**2行目**で買う
- 製造経費、1個20円で**2個まで**
